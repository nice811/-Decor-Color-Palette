# 装修配色工具站 - 测试方案

## 一、测试环境准备

### 1.1 环境配置
| 环境 | 环境变量 | 访问地址 |
|------|----------|----------|
| 开发环境 | VITE_DEV_DEBUG_LANG=true | http://localhost:5173/ |
| 生产环境 | VITE_DEV_DEBUG_LANG=false | npm run build:prod 构建后访问 |

### 1.2 前置条件
- Node.js >= 18.x
- npm install 已执行
- 开发服务器已启动 (npm run dev)

## 四、边界测试

### 4.1 异常场景
| 场景 | 操作 | 预期结果 |
|------|------|----------|
| 未知语言代码 | URL参数 ?lang=xyz | fallback到 en |
| 空语言参数 | URL参数 ?lang= | fallback到 en |
| 大小写混合 | URL参数 ?lang=ZH-CN | 正确匹配 zh-CN |

### 4.2 SEO合规性（生产环境）
| 检查项 | 验证方法 | 预期结果 |

| hreflang标签 | 检查HTML源码 | 无 zh-CN 条目 |
| meta标签 | 检查HTML源码 | 无中文描述 |
| sitemap | 检查sitemap.xml | 无 /zh-CN/ 路径 |
